Lenguajes Utilizados

Estilos

La parte de estilos esta basado en CSS, en ingles "Cascading Style Sheets" o hojas de estilo en cascadas, se utiliza para mejorar la version visual de la pagina web ademas de todos los procesos que pueden ser customizados, se uso una libreria llamada bootstrap en conjunto.

HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web.



¿Porque una E-Biblioteca?

Vi la necesidad de los alumnos de encontrar material y fuentes importantes en un solo lugar sin tener que recorrer varias paginas y lo mas simple posible con una interfaz amigable y fluida. La idea tambien surge de la creacion de anexos para aquellos alumnos que quieran informarse y educarse aparte lo que exige la curricula.



Herramientas Utilizadas

Las herramientas mas importantes para la realizacion del proyecto fueron, visual studio code para la creacion del codigo, Gimp para la edicion de imagenes, github para el versionado y backups, y sistema operativo linux en el cual se realizo todo el proyecto (Ubuntu 20.04 respectivamente).
